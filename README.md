# GUIbennar-REVIVED

A community-maintained revival of GUIbennar for Europa Universalis IV and Anbennar.

## About

GUIbennar is a user interface overhaul for the Anbennar total conversion mod for Europa Universalis IV. The original project introduced numerous improvements to the game's interface, making information easier to access while better accommodating Anbennar's unique mechanics.

Over time, updates to both Europa Universalis IV and Anbennar caused compatibility issues that left parts of GUIbennar broken or non-functional.

This project aims to restore and maintain GUIbennar for modern versions of EU4 and Anbennar while preserving the original vision of the mod.

## Goals

The primary objectives of this project are:

* Restore compatibility with current versions of Europa Universalis IV.
* Restore compatibility with current versions of Anbennar.
* Fix broken or missing interface elements.
* Resolve GUI errors and outdated references.
* Preserve the original look and feel of GUIbennar wherever possible.
* Document changes to make future maintenance easier.
* Provide a stable foundation for future improvements.

This project is focused on compatibility and maintenance first. Large-scale redesigns are not currently planned unless necessary to support newer game versions.

## Development Philosophy

The purpose of this repository is not to replace the original mod, but to keep it functional for the community.

Whenever possible:

* Original GUIbennar features will be preserved.
* Changes will be kept minimal and targeted.
* Compatibility fixes will take priority over redesigns.
* Contributions should aim to maintain the spirit of the original project.

## Current Target Versions

* Europa Universalis IV: 1.37.5
* Anbennar: Inheritors of Insyaa

These targets may change as new updates are released.

## Contributing

Contributions, bug reports, and testing feedback are welcome.

When reporting issues, please include:

* EU4 version
* Anbennar version
* A description of the issue
* Relevant screenshots
* Relevant log entries if available
